新虎通り 東京の未来を紡ぐ道

戦後の幻の計画から、東京を代表するシンボルストリートへ。70年以上にわたる壮大な都市再生の物語を、インタラクティブに紐解きます。

物語の始まり

「マッカーサー道路」と呼ばれた幻の計画から、新虎通りの誕生まで。70年近い歳月をかけた変遷の道のりを、インタラクティブな年表でたどります。キーとなる出来事を選択して、その背景を詳しく見てみましょう。

再開発の全体像

新虎通りの再開発は、森ビルが主導する虎ノ門ヒルズだけではありません。複数のデベロッパーが関わった公民連携の巨大プロジェクトです。下の図から街区を選択し、各プロジェクトの詳細をご覧ください。
